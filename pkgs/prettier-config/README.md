# @percolate/prettier-config

Percolate's [Prettier](https://prettier.io) config.

## Install

```bash
$ yarn add @percolate/prettier-config --dev
```

## Usage

Edit `package.json`

```jsonc
{
    // ...
    "prettier": "@percolate/prettier-config"
}
```
